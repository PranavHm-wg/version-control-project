# Task Tracker

## Overview
A minimal task tracker used to demonstrate Git & GitHub workflows for the assignment.

## Installation
1. `git clone git@github.com:your-user/version-control-project.git`
2. `cd version-control-project`
3. `python3 -m venv venv && source venv/bin/activate`
4. `python main.py`

## Features
- Phase-based Git demo
- CI workflow (simple)
- Docs and contributing notes

## Usage
`python main.py`
